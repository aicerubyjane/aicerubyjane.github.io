About Me

I build from small ideas, then keep sanding the rough edges until the interface makes sense.

I'm Aicerubyjane, a web developer and visual designer based in Indonesia with an interest in frontend development, web design, UI/UX, digital imaging, and 3D visual work.

I like the part where a page stops being a mockup and starts behaving like a real product: navigation, empty states, filtering, responsive layout, readable data, motion, visual assets, and small interaction details.

I started by building small academic and practice projects, then continued improving through web apps, landing pages, database design, interactive browser-based experiences, Figma layouts, Adobe edits, and Blender model exploration. Through these projects, I learned how to structure pages, manage data, design user flows, and connect visual direction with real functionality.

I'm currently improving my skills in HTML, CSS, JavaScript, PHP, Laravel, MySQL, Figma, Adobe tools, and Blender while building more polished projects for my portfolio.
