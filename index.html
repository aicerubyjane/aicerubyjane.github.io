About Me

Turning ideas into simple, useful, and visually clear websites.

I'm Aicerubyjane, a web developer based in Indonesia with an interest in frontend development, web design, and UI/UX.

I enjoy turning ideas into simple, functional, and visually clean websites. My work focuses on building responsive layouts, creating clear user interfaces, and developing practical web features that are easy to use.

I started by building small academic and practice projects, then continued improving through web apps, landing pages, database design, and interactive browser-based experiences. Through these projects, I learned how to structure pages, manage data, design user flows, and connect visual design with real functionality.

I'm currently improving my skills in HTML, CSS, JavaScript, PHP, Laravel, and MySQL while building more polished projects for my portfolio.
